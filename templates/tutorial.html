Introduction

Welcome to Object causal model labeler. The labeler will help you create a causal model for a furniture piece that can help the robot better understand how to assemble them. We will explain the interface and show you examples of correct causal model

What is Object Causal Model?

How to use the interface?

Step1: Associate object part with it's properties.

We provides you a list of object parts' names and a list of random properties. Click "Add object's properties" and click the dropdown menu to select the object with it's associated properties. Some parts can have more than one properties so you can click "Add object properties" multiple times untill you think you've add all the properties. After you're done, click "Submit" to finish.

Step2:Add object causal model

In this step, you need to complete a sentence using the list of words we provides to construct a causal relationship between object's properties to it's latent variables. We provides a list of property words, key words(logic and extend words) and latent words. Click "Add Causal Rule" to add a new causal rule. Click "Add property", "Add keyword" or "Add latent" to add a field that has a dropdown menu and select the word you want. After you complete your sentence, you can click "Add Causal Rule" again to add a new rule. After you finish adding all the causal rules, click "generate graph" and the website will display you with the causal graph you generated. Click "Submit" if you think the causal information is correct.
